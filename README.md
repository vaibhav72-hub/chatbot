# Medicine and Injury Detection Chatbot

A simple Python-based chatbot that provides information about **medications** and **injuries**.  
It uses [spaCy](https://spacy.io/) for natural language processing to detect keywords in user input and return relevant details.

---

## 🚀 Features
- Detects medications (e.g., paracetamol, ibuprofen, amoxicillin).
- Detects injuries (e.g., sprain, fracture, burn).
- Provides:
  - Usage / Description
  - Side effects / Treatment
  - Dosage (for medications)
- Interactive command-line chatbot interface.

---

## 🛠️ Requirements
- Python 3.7+
- spaCy library
- English language model (`en_core_web_sm`)
